# SmartStool
##### freelancehunt project

### Requirenments
* NodeJS >= 4.x

### Install node modules
```sh
$ git clone https://github.com/saynschuman/smartstool.git
$ npm install
```

### Install bower components
```sh
$ bower install
```

### Start server
```sh
$ npm start
```

### Start client
In browser:
[http://localhost:HOST](http://localhost:HOST)
